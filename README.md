# AmneziaWG Monitor

Веб-интерфейс для мониторинга и управления AmneziaWG VPN с авторизацией, SSL, сортировкой и выгрузкой конфигов.

## Возможности

- 📊 **Мониторинг в реальном времени** — список всех клиентов, статус подключения, трафик
- 🔐 **HTTP Basic Auth** — защита паролем
- 🔒 **SSL/HTTPS** — автоматический сертификат Let's Encrypt
- ✏️ **Редактирование имён** — клик по имени, сохранение в JSON
- 📥 **Выгрузка конфигов** — кнопка для скачивания конфига любого клиента
- 🔽 **Сортировка** — по IP адресу, полученному и отправленному трафику
- 🔄 **Автообновление** — каждые 30 секунд
- 🎨 **Тёмная тема** — адаптивный дизайн

## Требования

- Docker с запущенным контейнером AmneziaWG (имя контейнера: `amnezia-awg`)
- Домен, направленный на сервер (для SSL)
- Открытые порты: 80 (для Let's Encrypt), 9871 (веб-интерфейс)
